HTML

  • The head element contains information about the webpage.
  • The body element contains the visible content of the webpage.
  • HTML is a semantic language: many elements in an HTML docutment are named for what kind of content they contain.
  • Elements in the head element are meta-data that is mostly invisible to the user.
  • Elements in the body element contain structure based on the type of element. This line of text is a list item (li) inside of an unordered list (ul) in side of a section.

GIT

  • Here are some often-used git commands entered on the command line:
    • git status: checks what branch we are currently on
    • git checkout -b [branch-name]: creates a new branch and switches to it
    • git add -A: stages all the edited (or newly added) files to be ready for a commit
    • git commit -m "[commit notes]": commits all staged files to the current branch, and includes the given notes
  • After your work on a branch is complete, file a "Pull Request". A Pull request is a process whereby your code changes can be viewed by other team members and approved, before being merged into the main branch.
